# Store API

API REST desenvolvida com Laravel para fins de estudo e evolução em desenvolvimento backend.

## Objetivos

- Aprender a desenvolver APIs REST
- Aplicar arquitetura em camadas (Controller, Service e Repository)
- Utilizar Laravel Sanctum para autenticação
- Desenvolver boas práticas de organização e versionamento

## Tecnologias

- PHP
- Laravel
- MySQL
- Laravel Sanctum
